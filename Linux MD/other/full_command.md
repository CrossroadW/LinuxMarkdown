# 常用的命令及其全称
以下是一些常见的Linux命令及其全称：
1.  ls - list：列出目录中的文件和子目录
2.  cd - change directory：改变当前工作目录
3.  pwd - print working directory：显示当前工作目录的路径
4.  mkdir - make directory：创建一个新目录
5.  rm - remove：删除一个文件或目录
6.  cp - copy：复制一个文件或目录到另外一个位置
7.  mv - move：移动一个文件或目录到另外一个位置
8.  touch - touch：创建一个新文件或更新已有文件的时间戳
9.  cat - concatenate：连接文件并打印到标准输出设备上
10.  grep - global regular expression print：在文本文件中查找指定的字符串
11.  find - find：搜索文件系统中符合条件的文件
12.  chmod - change mode：更改文件或目录的权限
13.  chown - change owner：更改文件或目录的所有者
14.  ps - process status：显示当前运行的进程状态
15.  top - top of process：显示系统中使用最多CPU资源的进程
